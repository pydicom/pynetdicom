provider_reason¶

Indicates the reason for the abort. Allowed values are:

  • 0: reason not specified

  • 1: unrecognised PDU

  • 2: unexpected PDU

  • 4: unrecognised PDU parameter

  • 5: unexpected PDU parameter

  • 6: invalid PDU parameter value

int
