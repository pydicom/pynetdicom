Process the Presentation Contexts as an Association Requestor.

The Acceptor has processed the Requestorâs presentation context definition list and returned the results. We want to do two things:

  • Process the SCP/SCU Role Selection Negotiation items

  • Return a nice list of PresentationContext with the Results and original Abstract Syntax values to make things easier to use.

PresentationContextItemRQ

  • Presentation context ID

  • Abstract Syntax: one

  • Transfer syntax: one or more

PresentationContextItemAC

  • Presentation context ID

  • Result: one of 0x00, 0x01, 0x02, 0x03 or 0x04

  • Transfer syntax: one, not to be tested if result is not 0x00
