pynetdicom._config.USE_SHORT_DIMSE_AET¶

pynetdicom._config.USE_SHORT_DIMSE_AET: bool = True¶

Use short AE titles in DIMSE messages.

New in version 1.5.

If False then elements with a VR of AE in DIMSE messages will be padded with trailing spaces up to the maximum allowable length (16 bytes), otherwise no padding will be added.

Default: True

Examples

>>> from pynetdicom import _config
>>> _config.USE_SHORT_DIMSE_AET = False
