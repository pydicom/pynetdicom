Verification Service Class¶

The Verification Service Class defines a service that verifies application level communication between peer DICOM Application Entities. This verification is accomplished on an established association using the C-ECHO DIMSE-C service.

Supported SOP Classes¶

UID SOP Class
1.2.840.10008.1.1 VerificationSOPClass

Statuses¶

C-ECHO Statuses¶

Code (hex) Category Description
0x0000 Success Success
0x0122 Failure Refused: SOP Class not supported
0x0210 Failure Duplicate invocation
0x0211 Failure Unrecognised operation
0x0212 Failure Mistyped argument
