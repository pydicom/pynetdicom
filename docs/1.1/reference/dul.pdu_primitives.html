Service Primitives (pynetdicom.pdu_primitives)¶

Protocol Data Units (PDUs) are the message formats exchanged between peer application entities within a layer. A PDU consists of protocol control information and user data and is constructed using mandatory fixed fields followed by optional variable fields that contain one or more items and/or sub-items.

The DICOM Upper Layer Services are:

  • A-ASSOCIATE
  • A-RELEASE
  • A-ABORT
  • A-P-ABORT
  • P-DATA
